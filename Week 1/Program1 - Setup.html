Course Name : Single Page Applications with AngularJS

Setup & Requirements

1. Install Git and setup Github Repo for this Course

2. Install NodeJS for accessing npm

3. Using npm install browser-sync i.e npm install -g browser-sync

4. Setup Github Pages for the Repo, so that it can be served from the Github Repo

5. Use Browser-Sync to serve the created web pages i.e browser-sync --server --directory --file "**/*"
